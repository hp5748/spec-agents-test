# 接口规范

## API 端点

### 基础信息
- **Base URL**: `http://localhost:8000`
- **Content-Type**: `application/json`
- **字符编码**: UTF-8

## 对话接口

### POST /api/chat
普通对话接口，等待完整响应。

**请求体**:
```json
{
  "message": "用户消息内容",
  "session_id": "会话标识（可选）"
}
```

**响应**:
```json
{
  "session_id": "会话标识",
  "intent": "意图代码",
  "intent_name": "意图名称",
  "response": "回复内容"
}
```

**意图代码映射**:
| 代码 | 名称 | 说明 |
|-----|------|------|
| order_query | 订单查询 | 查询订单状态 |
| product_consult | 商品咨询 | 商品推荐 |
| complaint | 投诉处理 | 创建工单 |
| general_qa | 通用问答 | 常见问题 |

### POST /api/chat/stream
流式对话接口，使用Server-Sent Events (SSE)。

**请求体**: 同 `/api/chat`

**响应格式**: `text/event-stream`

**事件类型**:
```
data: {"type": "intent", "intent": "order_query", "intent_name": "订单查询"}

data: {"type": "content", "content": "正在查询..."}

data: {"type": "done"}
```

**事件字段说明**:
| type | 说明 | 附加字段 |
|------|------|---------|
| intent | 意图识别结果 | intent, intent_name |
| content | 内容块 | content |
| done | 流式结束 | - |
| error | 错误信息 | content |

## 会话管理

### GET /api/history/{session_id}
获取对话历史。

**响应**:
```json
{
  "session_id": "会话标识",
  "history": [
    {"role": "user", "content": "用户消息"},
    {"role": "assistant", "content": "助手回复"}
  ]
}
```

### DELETE /api/history/{session_id}
清空对话历史。

**响应**:
```json
{
  "message": "会话 xxx 的历史已清空"
}
```

### POST /api/session/new
创建新会话。

**响应**:
```json
{
  "session_id": "新生成的会话ID"
}
```

## 前端集成示例

### SSE 流式请求 (JavaScript)
```javascript
const response = await fetch('/api/chat/stream', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    message: '你好',
    session_id: 'test_session'
  })
});

const reader = response.body.getReader();
const decoder = new TextDecoder();

while (true) {
  const { done, value } = await reader.read();
  if (done) break;

  const text = decoder.decode(value);
  // 解析 SSE 格式: "data: {...}\n\n"
  const lines = text.split('\n');
  for (const line of lines) {
    if (line.startsWith('data: ')) {
      const data = JSON.parse(line.slice(6));
      // 处理 data.type, data.content 等
    }
  }
}
```
