# 订单查询工具 API

## 工具名称
`order_query`

## 功能
查询订单状态和物流信息

## 方法签名
```python
order_query._run(order_id: str) -> str
```

## 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| order_id | str | 是 | 订单号，格式：ORD + 10位数字 或 8位纯数字 |

## 返回值

成功时返回订单信息字符串，包含：
- 订单状态
- 物流信息
- 预计送达时间

## 错误处理

- 订单不存在：返回 "未找到订单"
- 格式错误：返回 "订单号格式不正确"

## 示例

```python
result = order_query._run("ORD1234567890")
# 输出: "订单 ORD1234567890 状态：已发货，预计3天内送达"
```
