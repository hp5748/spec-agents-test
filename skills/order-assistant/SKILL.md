# Order Assistant 技能

> **模板说明**: 本技能作为开发新技能的模板参考

订单查询服务，帮助用户查询订单状态。

---

## 目录结构

```
order-assistant/
├── SKILL.md              # 本文件 - 元数据和核心指令
├── scripts/
│   └── executor.py       # 技能执行器 (必需)
├── references/           # 参考资料 (可选)
└── assets/               # 模板资源 (可选)
```

---

## 功能说明

- 根据订单号查询订单状态
- 处理订单相关问题咨询

---

## 触发条件

当用户意图为 `order_query` 时自动触发。

### 订单号格式

- `ORD` + 10位数字：如 ORD1234567890
- 8位纯数字：如 12345678

---

## 执行流程

1. 从用户输入中提取订单号
2. 调用 `order_query` 工具查询
3. 格式化并返回结果
